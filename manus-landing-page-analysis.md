- **3-panel layout**: Sidebar (expanded) + Main Content Area + (No preview on landing)
- **Dark theme**: Consistent with task view

- **Left**: "manus" logo + collapse button
- **Center**: "Manus 1.6" dropdown
- **Right**:

  - Lightning icon with "86,301" (credits/points)
  - User avatar with small indicator

**Footer Icons:**

### Main Content Area (Center)
**Hero Section:**
- Large centered text: "What can I do for you?"

  - "+" button on left
  - Tool icons (GitHub, Google Drive, Google, +2 more)

**Quick Actions Row:**
- "Create slides" button with icon
- "Build website" button with icon
- "Develop apps" button with icon
- "Design" button with icon
- "More" button

**Feature Cards/Carousel:**
- Card showing: "Use Nano Banana Pro to create slides"
  - Subtitle: "Design pro-level slides with superior layouts and imagery instantly."
  - Preview image showing "Nano Banana" branding
- Pagination dots below (3 dots - indicating carousel)

### Key Observations

1. **This is a LANDING/HOME page** - not a task conversation view
2. **No chat messages** - just a prompt to start
3. **Quick action buttons** for common tasks
4. **Feature showcase carousel** at bottom
5. **Sidebar is the same** as in task view
6. **Header is simpler** - no Preview/Share/Published buttons (those are task-specific)

### Differences from Task View

| Element | Landing Page | Task View |
|---------|-------------|-----------|
| Main content | "What can I do for you?" prompt | Chat conversation |
| Header buttons | Minimal (notifications, credits, user) | Full set (Preview, Share, Published, etc.) |
| Input area | Large centered input | Bottom-fixed chat input |
| Quick actions | Visible (Create slides, Build website, etc.) | Not visible |
| Feature cards | Carousel showing features | Not visible |
| Preview panel | Never shown | Toggleable |
